`utils` package provides a set of utility functions used by vinxi's core packages.

Originally based in [oxy/utils](https://github.com/vulcand/oxy/master/tree/utils).

## Installation

```bash
go get -u gopkg.in/vinxi/vinxi.v0/utils
```
